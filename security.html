Module 03

Ports

A port is a number that identifies a specific process or service on a computer.
They work alongside IP addresses:

# Type Of Ports
Well-known ports (0–1023)
Reserved for common services. Examples:
80 → HTTP (web traffic)
443 → HTTPS (secure web traffic)
22 → SSH (secure shell)
25 → SMTP (email sending)
⚙️ How Ports Work
  • When you visit a website, your browser connects to the server’s IP address on port 443 (HTTPS)
  • The server responds back to your computer’s ephemeral port (a temporary one in the dynamic range).
  • This way, multiple applications can share the same network connection without confusion.

Payload Delivery

A payload is the actual exploit code executed on a target system. Reverse shells are the standard mechanism for establishing a persistent remote connection.

// Netcat Listener (Attacker Machine)
