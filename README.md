# GK // Portfolio V2

A highly interactive, premium editorial-style developer portfolio built with raw HTML, vanilla CSS, and GSAP. 
Designed with a "Newspaper meets Hacker Terminal" aesthetic, featuring smooth scroll-driven animations, an integrated OS-style terminal overlay, and high-end typography.

## Features
- **GSAP Choreography:** Intricate, timeline-driven page loaders and scroll reveals without relying on heavy frameworks.
- **Editorial Design System:** A vintage newspaper layout ("The Daily Ledger" style) paired with monospace "hacker" elements.
- **Integrated Terminal:** A functional overlay terminal mimicking a Linux OS environment with custom commands.
- **Zero CSS Frameworks:** Built purely with Vanilla CSS for total styling control.
- **Performance First:** Uses `requestAnimationFrame`, `IntersectionObserver`, and GSAP for buttery smooth 60fps animations.

## Architecture
- `index.html` - The main portfolio showcase (Hero, About, Projects, and Newspaper Footer).
- `security.html` - A dedicated internal page displaying the "Bug Bounty / Security Hall of Fame" with scroll-triggered data reveals.
- `script.js` - The master controller for GSAP animations, terminal logic, and global interactions.

## Deployment
This project is configured to run flawlessly on **GitHub Pages**. All internal links are relative, and a custom `404.html` handles any routing fallbacks natively.
